物件
    此處先演示 簡易的物件宣告，因為還可以透過 Interfaces(介面)方式 ，類別(type) 方式來宣告
    這些會在後面在一起示範
    
    ==> let 物件名稱 : {
            "屬性名稱":屬性類別,
            "屬性名稱":屬性類別,
            "屬性名稱":屬性類別,
            "屬性名稱":屬性類別,
        }



    let Person :{
        "name":string,
        "age" : number,
        "gender" : string,
        "country" : string,
    }
 
    Person={
        "name":"Jack",
        "age" : 18,
        "gender" : "male",
        "country" : "ROC",
    } //編譯正常

    可讀屬性
        有些屬性，我們希望它可以出現或是不需要( 例如比較隱私的資訊 )，那麼我們可以在屬性上面添加一個問號 ?
        此時，滑鼠屬移動到 添加?的屬性上，會顯示 它可以接受你原本綁定的屬性 且 會再多綁一個可以接收的 undefined
    ==> let 物件名稱 : {
            "屬性名稱" : 屬性類別,
            "屬性名稱"? : 屬性類別, //添加問號後，該屬性可以賦值也可以不需要
            "屬性名稱" : 屬性類別,
            "屬性名稱" : 屬性類別,
        }
